# Reality Fusion
## Managed by [Zernov](https://www.youtube.com/@zernovtech)
[README.md на русском](./READMERUS.md)

## About
Unity mixed reality application that working only on a Android phone.
Includes Hand Tracking, Stereovision, WebBrowser, PC streaming, virtual cinema and many interesting things!

# Let's begin!
## If you just want to use it...

1. Open [releases](https://github.com/ZernovTechno/AR/releases) page and download [latest](https://github.com/ZernovTechno/AR/releases/latest) version.
2. Run APK and install it on the phone.
3. Choose tracking type:
   Center point (cardboard standart)
   or hand tracking
4. Have fun

## If you want to compile that...
First, [download and install Unity 6](https://unity.com/releases/unity-6). 

1. Then download Reality Fusion sources.
2. **BEFORE OPENING PROJECT** download additional packages from [url](https://mega.nz/file/WqZ2ACaR#6n6de0GPABapOfzS8Vr_B48c5cc9L9SwOouPFRzc7nA), and drop folder from zip to ./Assets
3. Open sources as project in Unity Hub.
4. Build application in Unity.
